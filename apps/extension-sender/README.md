# EasyTransfer Browser Sender Extension

## Load

1. Open Chrome/Edge: `chrome://extensions`
2. Enable Developer mode
3. Load unpacked -> select this folder

## Usage

1. Select files
2. Click **Prepare Frames**
3. Click **Start Playback**
4. Use Android scanner app to scan frames on screen

This sender applies size-based compression policy and emits source + repair symbols.
